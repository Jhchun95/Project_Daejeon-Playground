> ## 대전광역시 어르신 놀이터 최적 입지 선정
> 1. 이 프로젝트의 코드는 본인이 직접 코드를 개발한 것이 아니고, 팀원이 작업한 코드임을 밝힘.  
>    팀원들이 어떤 방향으로 코딩해서 전처리하고, 진행했는지 본인이 얼마나 이해했는지 업로드한 것.  
> 2. 공공 데이터포털 사이트에서 공유하고 있는 데이터와 현장방문, 인터뷰를 통해 데이터를 종합하여 수집함.
